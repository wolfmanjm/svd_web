This is the code that serves https://svd.wolfman.com

It can take a .SVD file from any MCU and produce a database, which then can be accessed from the web.

It shows a list of MCUs that have been loaded on the left. 
Selecting one of them shows the list of Peripherals for that MCU in the main window.
Selecting a peripheral will then show a lit of all the registers for that peripheral.
Selecting a register will then show a list of all the bit fields for that register.
